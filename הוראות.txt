חשוב:
  אל תפתח index.html ישירות — חייב שרת מקומי.

כפתורים בממשק:
  + פרויקט    — יצירת פרויקט חדש (עולמות + נתיב + משימה)
  עולמות      — עריכת שם, פועל וצבע לכל עולם
  חוטים       — עריכת רשת החוטים + תוויות בעברית
  פורטפוליו   — תצוגה מקדימה + סנכרון (both/from/to)
  📁 בפרויקט   — פתיחת תיקייה ב-Explorer
  MCP         — שרת MCP אמיתי + manifest (bridge-mcp-server.py)
  רשימה       — כל הפרויקטים בפאנל
  YAML        — הצג + הורד bridge-state.yaml (כולל threads)
  שמור        — שמירה + COMPARTMENT_MAP + bridge-export + BRIDGE_MCP
  ⚙           — הגדרות bridge-config.json (GitHub + Netlify tokens)
  📡           — רענון GitHub + לייב + Netlify Analytics (7d/30d)
  מצב שקוף    — רק רשת החוטים + תוויות בעברית (מקש T)

פס סטטיסטיקות (מתחת לעולמות):
  פרויקטים · פעילים · חוטים · לייב · צפיות Netlify (30 יום)
  מתעדכן מ-/api/dashboard

Netlify Analytics:
  1. העתק bridge-config.example.json → bridge-config.json
  2. הוסף netlify.token (Netlify > User settings > Personal access tokens)
  3. כל אתר דורש Netlify Analytics בתוכנית
  4. לחץ 📡 לרענון — badge NF מציג צפיות/30י לכל פרויקט

מצב שקוף — רק חוטים:
  כפתור בתחתית או מקש T
  רקע שקוף, עולמות מעומעמים, 10 חוטים זוהרים
  כל חוט מציג תווית בעברית (מ-bridge-data.json, שדה שלישי)
  דוגמה: "אותה שאלה על AI — מבפנים ומבחוץ"
  הטקסט המרכזי נעלם אחרי ~4 שניות — נשארות רק התוויות

שקיפות אמיתית על שולחן העבודה (overlay):
  חוטים שקוף.bat (על שולחן העבודה)
  או: BridgeOS\הפעל overlay שקוף.bat
  WebView2 שקוף מלא מסך — רואים טפט + חוטים זוהרים
  דורש: pywebview (pip install pywebview) + שרת Bridge (8787)
  Rainmeter: התקן Rainmeter חוטים.bat → כפתור 🧵 על המסך
  סגירה: Esc · כפתור ✕ סגור שכבה · Alt+F4
  סגור overlay.bat — כפיית סגירה

חיבור MCP (Claude / Cursor):
  1. py -3.12 -m pip install mcp
  2. הפעל שולחן עבודה.bat (חובה — פורט 8787)
  3. הוסף ל-MCP settings את התוכן מ-mcp-config.json
  4. כלים: bridge_list_projects, bridge_open_terminal, bridge_metrics...

API (מקומי):
  GET  /api/dashboard      — סטטיסטיקות
  GET  /api/mcp            — MCP manifest ל-Claude
  GET  /api/export         — bridge-export.json
  GET  /api/metrics        — GitHub + לייב + Netlify deploy + pageviews
  GET  /api/netlify        — סיכום Analytics (7d/30d לכל אתר)
  GET  /api/sync-portfolio/preview — תצוגה מקדימה לפני סנכרון
  POST /api/sync-portfolio?direction=both — סנכרון (both|from|to)
  GET  /api/folder/{id} — פתיחת תיקיית פרויקט
  GET  /api/ecosystem   — סטטוס קבצים אחרונים + מפת סטטוס
  GET  /api/health      — בדיקת בריאות: שרת, קיצורים, MCP, כלים ונתיבי פרויקטים
  GET  /api/recent/{id} — קבצים אחרונים לפי נתיב פרויקט
  GET/POST /api/config  — קריאה/שמירת bridge-config.json
  GET  /open/{id}          — טרמינל + BRIDGE_SESSION.md
