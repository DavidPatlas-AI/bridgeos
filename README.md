## 🎯 מה זה

ניהול עשרות פרויקטים מקבילים דורש 'מוח שני' — BridgeOS הוא סביבת עבודה מקומית עם מפת 3D של כל הפרויקטים, חוטים בעברית, וסוכן AI לכל פרויקט.

## 🛠️ טכנולוגיה

`HTML5` · `Python` · `MCP`

## 📦 הרצה

דורש שרת מקומי — הפעל `שולחן עבודה.bat` ופתח `http://127.0.0.1:8787/` (לא לפתוח `index.html` ישירות).
